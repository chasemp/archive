Phabricator conduit testing via CLI

Wednesday, December 24, 2014 » conduit, phabricator

Search for a user by username:

echo '{"usernames": ["admin"]}' | \
/srv/phab/arcanist/bin/arc call-conduit user.query

Get task information by id:

echo '{"task_id": "9"}' | \
/srv/phab/arcanist/bin/arc call-conduit maniphest.info

Add comments to a task:

echo '{"id": "9", "comments": "this is comment1"}' | \
/srv/phab/arcanist/bin/arc call-conduit maniphest.update

Create a task:

echo '{"title":"foo 123"}' | \
/srv/phab/arcanist/bin/arc call-conduit maniphest.createtask

Create a task while supplying a custom field value:

echo '{"title":"foo 234", "auxiliary":{"std:maniphest:external_id":"789"}}' | \
/srv/phab/arcanist/bin/arc call-conduit maniphest.createtask

If you want more debugging information on any call you can add –trace.
