cisco ios-xe history size

Wednesday, November 21, 2012 » cisco, IOS

Preamble

Changing your session buffer for better command navigation.

Per VTY

History buffer size in IOS is weird. It's per VTY. So you have to
change it for all virtual terminals in service.

Your VTY configuration probably looks something like:

line vty 0 4
 transport input ssh
 escape-character 3
line vty 5 15
 transport input ssh

Change settings for VTY's

line vty 0 15
 !256 is the max
 history size 256

Alias

alias exec hg show history | include

Looks like:

<your_device>#hg conf
    hg conf
