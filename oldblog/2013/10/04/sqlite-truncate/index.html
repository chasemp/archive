sqlite truncate

Friday, October 04, 2013 » sqlite, truncate

SQLite does not seem to support the truncate command. This is hacky way to do it.

Getting the schema for a table name…



SQLite version 3.7.7 2011-06-25 16:35:41

Enter ".help" for instructions

Enter SQL statements terminated with a ";"

sqlite> .schema

Let's say this is part of our schema…

CREATE TABLE MYTABLE(myfield TEXT NOT NULL,
                       datetime timestamp);

Drop all rows in a table

Delete from MYTABLE

Reset autoincrement counter if applicable

DELETE FROM SQLITE_SEQUENCE WHERE name='MYTABLE';
