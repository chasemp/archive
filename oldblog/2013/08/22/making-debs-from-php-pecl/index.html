making debs from php pecl

Thursday, August 22, 2013 » deb, debian, linux, pecl, php

Preamble

Bending PECL to your deb building will.

Installing Packages

apt-get install dh-make-php \
                php5-dev \
                build-essential \
                libmagic-dev \
                debhelper \
                buildpackage

Example 1


git clone https://github.com/php-test-helpers/php-test-helpers.git
cd php-test-helpers/
phpize
./configure
make
make test
make install

Example 2


pecl download phpunit/test_helpers
dh-make-pecl --maintainer "Your Name " --arch amd64 test_helpers-1.1.0.tgz 
cd php-test-helpers-1.1.0/ && ./debian/rules binary
