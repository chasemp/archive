Preamble

mcollective-isms.

Find all hosts

Create an shortcut

/usr/local/bin/mc-find-hosts


#!/bin/bash
if [ "$1" == "" ]; then
    /usr/bin/mco find --wc mcollective
else
    /usr/bin/mco find $@


mc-find-hosts

Find host by name matching

mc-find-hosts --with-id /foo/

Find hosts by puppet class

/usr/sbin/mc-find-hosts --with-class <class_name> | sort

Query service status by class

mco service --with-class /apache/ apache2 status

Getting stats about the service itself

mco rpc rpcutil daemon_stats

Restarting a service

mco service --with-identity <host> <service> restart

Find agents directory

sudo find / -name rpcutil.ddl -exec dirname {} \;

Query a fact

mco facts <fact_name>

Query Puppet Status and get JSON in return

mco rpc puppet status -j
