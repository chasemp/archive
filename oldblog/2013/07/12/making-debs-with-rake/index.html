making debs with rake

Friday, July 12, 2013 » debian, rake, ruby

Preamble

Some Rake files have built in logic to make a debian binary.

Mcollective

A good example:

git clone https://github.com/puppetlabs/marionette-collective
cd marionette-collective/

Getting Packages

aptitude install rake \
build-essential 
rubygem \
pdebuild-cross \
devscripts \
fakeroot \
dpatch

Making Debian Package

rake deb

Looks Like


rm -r build
rm -r doc
